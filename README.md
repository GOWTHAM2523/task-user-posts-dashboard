# React TypeScript User Posts Dashboard

A production-ready React application built with TypeScript, Vite, and TailwindCSS that displays users and their posts with full CRUD operations and infinite scroll pagination.

## Features

### Core Features
-  **Users List Page**: Display all users with name, email, and company
-  **User Detail Page**: View user information and their posts
-  **Infinite Scroll Pagination**: Load posts in batches of 5 with auto-scroll detection
-  **CRUD Operations**: Add, Edit, and Delete posts (local state only)
-  **Loading States**: Simulated network delay (400-700ms) with loading indicators
-  **Error Handling**: Comprehensive error states with retry functionality
-  **Responsive Design**: Mobile-friendly UI with TailwindCSS

### Bonus Features
-  **Search Functionality**: Filter posts by title on user detail page
-  **Context API**: Centralized state management
-  **Notifications**: Toast messages for post operations
-  **Accessibility**: Keyboard navigation and semantic HTML
-  **TypeScript**: Full type safety throughout the application

## Technology Stack

- **Frontend Framework**: React 18.2
- **Language**: TypeScript 5.2
- **Build Tool**: Vite 5.1
- **Routing**: React Router v6.22
- **Styling**: TailwindCSS 3.4
- **State Management**: React Context API
- **Data Fetching**: Custom hooks with simulated delay
